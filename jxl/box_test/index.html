This contains jxl images with various kinds of boxes, some with three junk bytes added after the codestream.
box type no junk with junk
0 Do not use box format at all, only pure codestream
1 Have a single codestream box, with its actual size given in the box
2 Have a single codestream box, with box size 0 (final box running to end)
3 Single codestream box, with another unknown box behind it
4 Have multiple partial codestream boxes
5 Have multiple partial codestream boxes, with final box size 0 (running to end)
6 Have multiple partial codestream boxes, terminated by non-codestream box
7 Have multiple partial codestream boxes, terminated by non-codestream box that has its size set to 0 (running to end)
8 Have multiple partial codestream boxes, and the first one has a content of zero length
9 Have a compressed exif box before a regular codestream box
